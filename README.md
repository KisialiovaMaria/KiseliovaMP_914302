# Face Recognition Access Control System

### Описание:
Система для контроля доступа на охраняемый объект с использованием распознавания лиц

#### Система позволяет:
- распознавать лица посетителей
- вести учет посещений
- управлять уровнями доступа сотрудников
- уведомлять о попытках несанкционированного доступа

#### Архитектурные решения
- Клиент-серверная архитектура для масштабируемости и многопользовательского доступа  
- Основные данные хранятся в БД для обеспечения целостности  
- Изображения вынесены в файловую систему для повышения производительности  
- Реализована система уведомлений для повышения безопасности
